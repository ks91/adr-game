# 通貨による交換
* 各々のプレイヤーは自分が保有する法定通貨と、マーケットが保有するアイテムを交換することが出来ます。この交換を「（法定通貨で）アイテムを買う」と呼びます。他のプレイヤーが保有するアイテムと直接交換することは出来ません。
* 各々のプレイヤーは自分が保有するADR通貨と、マーケットが保有するアイテムを交換することが出来ます。この交換を「（ADR通貨で）アイテムを買う」と呼びます。他のプレイヤーが保有するアイテムと直接交換することは出来ません。
* 各々のプレイヤーは自分が保有するアイテムと、マーケットが保有する法定通貨を交換することが出来ます。この交換を「（法定通貨で）アイテムを売る」と呼びます。他のプレイヤーが保有する法定通貨と直接交換することは出来ません。
* 各々のプレイヤーは自分が保有するアイテムと、マーケットが保有するADR通貨を交換することが出来ます。この交換を「（ADR通貨で）アイテムを売る」と呼びます。他のプレイヤーが保有するADR通貨と直接交換することは出来ません。
* 各々のプレイヤーは自分が保有する法定通貨と、マーケットが保有するADR通貨を交換することが出来ます。この交換を「ADR通貨を買う」または「法定通貨を売る」と呼びます。他のプレイヤーが保有するADR通貨と直接交換することは出来ません。
* 各々のプレイヤーは自分が保有するADR通貨と、マーケットが保有する法定通貨を交換することが出来ます。この交換を「ADR通貨を売る」または「法定通貨を買う」と呼びます。他のプレイヤーが保有する法定通貨と直接交換することは出来ません。

# 通貨による雇用、および解雇
* 各々のプレイヤーは自分が保有する法定通貨と、マーケットが保有する「人材」を交換することが出来ます。この交換を「（法定通貨で）人材を雇用する」と呼びます。他のプレイヤーが保有する人材と直接交換することは出来ません。
* 各々のプレイヤーは自分が保有するADR通貨と、マーケットが保有する「人材」を交換することが出来ます。この交換を「（ADR通貨で）人材を雇用する」と呼びます。他のプレイヤーが保有する人材と直接交換することは出来ません。
* 各々のプレイヤーは自分が保有する「人材」と、マーケットが保有する法定通貨やADR通貨を交換することは出来ません。自分の所属から外す場合は、無償で所属をマーケットに変更しなければなりません。この交換を人材を解雇する」と呼びます。

# 交換レート
* このゲームでは、1AD = 1USD です。
* ゲームを実行中のある時点から見て、あるアイテムと法定通貨が最後に交換された際の法定通貨の量を、そのアイテムの「（法定通貨による）価額」と呼び、単位はUSDです。
* ゲームを実行中のある時点から見て、あるアイテムとADR通貨が最後に交換された際の ADR通貨の量を、そのアイテムの「ADR通貨による価額」と呼び、単位はADです。

# 保有量
* プレイヤーが保有する法定通貨の初期値は 20,000,000USD です。
* ADR通貨はデブリ除去によってしか生じないので、各プレイヤーが保有する ADR通貨の量は、初期段階において 0AD です。
* マーケットが保有する法定通貨の初期値は 20,000,000USD × プレイヤーの数です。
* マーケットが保有するADR通貨の初期値は 0AD です。

# 資産総額
* あるプレイヤーが保有する全てのアイテムの法定通貨による価額の合計と、そのプレイヤーが保有する法定通貨および ADR通貨の価額の総額を、そのプレイヤーの「資産総額」と呼びます。
* マーケットが保有する全てのアイテムの法定通貨による価額の合計と、マーケットが保有する法定通貨および ADR通貨の価額の総額を、マーケットの「資産総額」と呼びます。
* 全プレイヤーが保有する「資産総額」とマーケットが保有する「資産総額」の合計を「世界の資産総額」と呼びます。
* 全てのプレイヤーの「通貨保有量」とマーケットの「通貨保有量」の合計を「（世界の）通貨保有量」または「マネー・ストック」と呼びます。
* 各ターンの最初に、「マネー・ストック」の量が「世界の資産総額」の10%程度になるように、マーケットの法定通貨の量は GM との間で移動されます。つまり、「マネー・ストック」の量が「世界の資産総額」の10%に満たない場合には、マーケットの法定通貨の量を増やし、10%を超える場合には、マーケットの法定通貨の量を減らします。マーケットの法定通貨の量は0が下限です。

# 法人税
* 最初のターンを第１ターンとして、数えた回数が4の倍数となるターン（このターンを年度末ターンと呼ぶ）の終了時に、あるプレイヤーの法定通貨による資産総額が、４回前のターンの終了時の資産総額に比べて増えている場合、増えた量に対して「一定の割合 (20%)」の法定通貨が、そのプレイヤーの所属から減らされ、減らされた量と同量が GM が保有する量の増加量となります。つまりプレイヤー所属から GM 所属に法定通貨が移動します。この移動そのものと、移動した量は、いずれも「法人所得税」または「法人税」と呼ばれます。
* この法定通貨の移動、すなわち「法人税の課税」は、全てのプレイヤーに対して、年度末ターンの終了時に実施されます。
* あるプレイヤーが保有する法定通貨の量が「法人所得税」よりも少ない場合、プレイヤーが保有する法定通貨の量はマイナスの値となります。マイナスの値になると利息が発生し、1ターンあたり、0.5%ずつ、その絶対値が増加します。
* 上記の「一定の割合」はある同一のターンにおいては全てのプレイヤーに対して等しい値ですが、ターンによって変動します。初期には20%程度に設定し、プレイヤー所属の法定通貨の増加率が、平均で4ターン当たり2%程度になるように調整する必要があります。

# ADR通貨の減価
* プレイヤーが保有するADR通貨は、1ターンあたり0.5%で減価します。
* マーケットが保有するADR通貨も、1ターンあたり0.5%で減価します。
